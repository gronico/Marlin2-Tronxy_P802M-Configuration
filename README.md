# Marlin2-Tronxy_P802M-Configuration
Marlin's Configuration files for Tronxy P802M 3D printer. 2 setup are available: STD and 12mm NPN capacitive bed level sensor (CPL).


- Version is the same as the latest tested Marlin 2 firmware (might work with newer).
- STD : for a standard out of the box Tronxy 802M printer
- CPL : For a P802M with a 12mm Capacitive Sensor (LJ12A3-4-Z/BX) for bed leveling (need a metal bed to work properly).
